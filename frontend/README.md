# Frontend

This directory contains the React frontend for the app.

Main commands:

- `npm install`
- `npm run dev`
- `npm run build`

Responsibilities:

- operator dashboard UI
- current market graph and live cycle view
- autopilot controls
- wallet and account views
- trade ticket UI
- AI command center and support chat

Key files:

- `src/App.jsx`: main app shell and screens
- `src/styles.css`: dashboard styling
- `vite.config.js`: Vite config
